## 系统介绍简介

一直想做一款面向中小企业简单便捷的进销存管理系统，于是利用空闲休息时间开始自己写一套简易进销存系统。如此有了企业进销存管理系统。当然，您也可以对它进行深度定制，以做出更强系统。同时支持移动客户端访问。系统会陆续更新一些实用功能。


## 核心功能
本项目基于RuoYi进行搭建，在若依基础上进行功能构建、数据连接。

 **仓库系统(开源)** 
1.  基础资料模块：包含客户、供应商、承运商基础信息资料。
2.  仓库设置模块：配置仓库存放的库区库位。
3.  商品管理模块：商品类别和商品信息管理。
4.  仓库管理模块：库存查询和单据精灵(可开退货单和消退单)等。
5.  进货管理模块：包含其他进货和收货作业功能。
6.  出库管理模块：包含其他出库和出库作业、缺货管制订单功能。

 **订单系统(未开源)** 
1.  采购管理模块：包含采购订单和请拔订单(需要设置单位关系)功能。
2.  销售管理模块：包含销售订单和受拔订单(需要设置单位关系)功能。
3.  基础资料：单位关系管理，可以同集团旗下不同分公司相互之间调拔，比如:索孚科技集团(总仓)、深圳企业配件公司(分仓)、长沙企业配件公司(分仓)，
总仓、分仓相互之间可以通过请拔订单和受拔订单进行调货，调货价格根据单位关系设定取成本价、批发价或销售价。
